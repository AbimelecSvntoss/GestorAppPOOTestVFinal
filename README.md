# App Gestor | TecNM Campus Monclova

Un entorno de desarrollo se basa en distintas fases tales como análisis, diseño, planificación, desarrollo y mantenimiento, por lo que al aplicarlas en la vida cotidiana o proyectos personales/universitarios,
debería haber una organización para llevarse a cabo. En este repositorio creamos una calculadora básica
con Programación Orientada a Objetos usando Herencia, Polimorfisno, Clases y demás para llevar esa teoría a la practica.
